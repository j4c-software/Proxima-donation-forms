# Proxima Donation Forms

A lightweight, plain HTML/CSS/JS donation form with preset amount selection and custom amount input.

## Features

- Preset donation amounts ($10, $25, $50, $100)
- Custom amount input
- Name, email, and optional message fields
- Clean, responsive card layout
- No dependencies — pure HTML, CSS, and JavaScript

## Getting Started

Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/mike-j4c/Proxima-donation-forms.git
cd Proxima-donation-forms
open index.html
```

## Project Structure

```
Proxima-donation-forms/
├── index.html   # Form markup
├── style.css    # Styles
└── script.js    # Amount selection & form submission logic
```
